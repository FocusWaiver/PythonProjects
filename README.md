# PythonProjects
Group of coding projects written in Python <br>
## Contents:
- ELISA Mapping Calculator <br>
   Output last sample well on a 96 well plate for each case, usually 75 or 150 samples per case.
- VariousQuizzer <br>
    2 modes, km to mi and c to f, and reports end score. 
- ReadingMusicQuizzer <br>
   Test user on Treble and Bass clef by randomly choosing a note and drawing it on the staff.
- GameScoreboard <br>
   Update a saved score and calculate the new score based on past round. For games like Gin Rummy, Uno.
- RandomWikiSummaryGenerator <br>
    Uses Wiki api to collect the lead summary from a random wiki article.
