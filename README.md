# PythonProjects
Group of coding projects written in Python
- **List of Projects:** <br>
- ELISA Mapping Calcultor
-   Output last sample well on a 96 well plate for each case, usually 75 or 150 samples. <br>
- VariousQuizzer
-   2 modes, km to mi and c to f, and reports end score. <br>
- ReadingMusicQuizzer
-   Test user on Treble and Bass clef by randomly choosing a note and drawing it on the staff. <br>
- GameScoreboard
-   Update a saved score and calculate the new score based on past round. For games like Gin Rummy, Uno. <br>
- RandomWikiSummaryGenerator
-   Uses Wiki api to collect the lead summary from a random wiki article. <br>
